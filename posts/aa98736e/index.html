Windows上Java的环境变量配置
发表于 2018-10-14| 更新于 2019-12-07|教程Windows
字数总计: 482|阅读时长: 1 分钟| 阅读量:

对于开发者来说，Java是电脑上必备安装的软件。安装Java不单单只是安装jdk就可以，还要配置Java的环境变量。这篇文章主要是介绍如何配置Java的环境变量。

电脑系统： Windows 10 1809
Java版本： 1.8.0_181

安装Java

第一步当然是安装Java文件。Java现在已经发现到Java11了，不一定要安装最新的版本，可以根据自己的需要安装对应的版本。
本文安装的 Java SE 8u181
下载Java可以到oracle的官网：https://www.oracle.com/technetwork/java/javase/downloads/index.html

新建Java_Home变量

右键This PC(即此电脑)，选择Properties(属性)选项。

点击左边栏的Advanced sysyem settings(高级系统设置)，点击下面的Environment Variables(环境变量)

点击System variable(系统变量)下的New(新建)。

Variable name(变量名) 填爲 Java_Home
Variable value(变量值) 爲Java的安装路径。例如我的是 C:\Program Files\Java\jdk1.8.0_181
然后点击OK(确定)

编辑 Path变量

在System variable(系统变量)下找到Path并双击。
在打开的窗口中点击右边的编辑文本。
把 %Java_Home%\bin;%Java_Home%\jre\bin; 複製到 Variable value(变量值) 的开头并保存。

新建 CLASSPATH

点击System variable(系统变量)下的New(新建)。

Variable name(变量名) 填爲 CLASSPATH
Variable value(变量值) 填爲 .;%Java_Home%\bin;%Java_Home%\lib\dt.jar;%Java_Home%\lib\tools.jar
然后点击OK(确定)

测试

上面已经把相关的环境变量给配置好了，接下来测试下是否配置正确。
打开CMD或者PowerShell

输入Java

接着输入 java -version

接着输入 javac

如果你的结果跟上面图片的结果一样或者相似的话，恭喜你，Java的环境变量配置已经成功了。
